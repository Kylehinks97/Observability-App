# DevPulse — Project Brief

## Purpose

Portfolio project built to target a fullstack role at Datadope (and similar observability/data companies).
Demonstrates: FastAPI, React + TypeScript + Tailwind, WebSockets, PostgreSQL + SQLAlchemy, LangChain AI, Docker, Pytest.

The project is a mini real-time infrastructure monitoring dashboard — essentially a toy version of what Datadope builds.
Showing this to them signals you understand their domain, which is rare and memorable.

---

## What It Does

- **FastAPI backend** generates fake infrastructure metrics (CPU, memory, API latency, error rate)
- **Saves snapshots** to PostgreSQL via SQLAlchemy every second
- **Streams live data** to the React frontend via WebSockets
- **React dashboard** displays real-time line charts (Recharts) and stat cards
- **AI chat panel** — user types "why did latency spike at 3pm?", LangChain queries recent DB rows and answers
- **JWT auth** with two roles: admin (can see all metrics) and viewer (read-only)
- **Dockerized** — single `docker-compose up` runs everything

## Key Code Concepts

### Fake metrics generator (backend/app/services/simulator.py)
```python
import random, time

def generate_metric():
    return {
        "timestamp": time.time(),
        "cpu": random.uniform(20, 95),
        "memory": random.uniform(40, 85),
        "api_latency_ms": random.uniform(50, 500),
        "error_rate": random.uniform(0, 5)
    }
```

### WebSocket endpoint (backend/app/api/routes/ws.py)
```python
from fastapi import WebSocket
import asyncio
from app.services.simulator import generate_metric

@app.websocket("/ws/metrics")
async def metrics_stream(websocket: WebSocket):
    await websocket.accept()
    while True:
        data = generate_metric()
        await websocket.send_json(data)
        await asyncio.sleep(1)
```

### React WebSocket hook (frontend/src/hooks/useMetricsSocket.ts)
```ts
import { useEffect, useState } from 'react'

export function useMetricsSocket() {
  const [metrics, setMetrics] = useState(null)

  useEffect(() => {
    const ws = new WebSocket('ws://localhost:8000/ws/metrics')
    ws.onmessage = (event) => setMetrics(JSON.parse(event.data))
    return () => ws.close()
  }, [])

  return metrics
}
```

### AI chat service (backend/app/services/ai_service.py)
```python
from langchain_openai import ChatOpenAI

def ask_about_metrics(question: str, recent_metrics: list) -> str:
    llm = ChatOpenAI()
    prompt = f"""
    You are an observability assistant. Here are the last 60 minutes of metrics:
    {recent_metrics}

    Answer this question: {question}
    """
    return llm.invoke(prompt).content
```

## Build Order

### Phase 1 — Backend foundations
1. FastAPI + `/metrics` REST endpoint returning fake data (no WebSockets yet)
2. PostgreSQL + SQLAlchemy model — save each metric snapshot to DB
3. JWT auth — `/login`, `/register`, protected routes, RBAC (admin/viewer)

### Phase 2 — Frontend
4. React + TypeScript + Tailwind scaffold, Login page with JWT
5. Dashboard with Recharts line charts, polling `/metrics/history` every 3s
6. Looks real-time. Pause here if short on time — this is already impressive.

### Phase 3 — Impressive additions
7. Swap polling for WebSocket using `useMetricsSocket` hook
8. AI chat panel — LangChain queries recent DB rows, answers natural language questions

### Phase 4 — Polish
9. Docker + docker-compose (postgres + backend + frontend)
10. Pytest for auth and metrics routes
11. Verify OpenAPI docs at `/docs` look clean (FastAPI generates these automatically)
